# Projeto Leitor-FTP

# Rodar o servidor de ftp
Na pasta "docker"
```
docker-compose -f docker-server-ftp.yml up
```

# Rodar o S3ninja
Na pasta "docker"
```
docker-compose -f docker-s3ninja.yml up
```

# Rodar o postgres
Na pasta "docker"
```
docker-compose -f docker-postgres.yml up
```

# Rodar o kafka
Na pasta "docker"
```
docker-compose -f docker-kafka.yml up
```
